# Seonrin Design System

## Brand
선린은 좋은 이웃이라는 의미를 가진 지역 중개사무소 브랜드다.
디자인은 과장된 럭셔리가 아니라, 조용한 신뢰·서류 확인·동네 상담의 안정감을 전달해야 한다.

## Visual Keywords
warm ivory
paper glass
soft local trust
documentary real-estate
calm consultation
not cyberpunk
not finance dashboard
not luxury hotel
not generic SaaS

## Core UI Rule
첫 화면은 안심감과 상담 진입.
drawer는 복잡한 상담을 차분히 고르는 안내서.
모달은 손님이 필요한 정보를 부담 없이 확인하는 설명창.

## Typography
Hero title: serif display
Body and UI: Korean sans
Hero title must not look like heavy system gothic.

## Color Tokens
ivory, warm paper, soft brown ink, muted blue accent.

## Glass Rule
All translucent surfaces must use:
- warm white background
- white border
- subtle inset highlight
- soft warm shadow
- blur

## Face Safe Rule
The hero portrait must not cover or damage eye readability.
Overlay must protect text readability but must not create a gray mask over the face.

## Drawer Rule
Drawer is not a settings panel.
Drawer is a consultation guide.
It should feel like a premium paper-glass advisory sheet.
